# Images Directory

Place your gym images here following this structure:

## Required Images

### Root images/
- `hero-bg.jpg` — Full-screen hero background (1920×1080px recommended)
  - Should be a dramatic, dark gym interior or athlete shot
  - Will have gradient overlay applied automatically

- `about-gym.jpg` — About section feature image (800×1000px recommended)
  - Portrait orientation works best
  - Gym interior, equipment area, or atmosphere shot

### images/trainers/
- `trainer-1.jpg` — Alex Rivers (600×800px, portrait)
- `trainer-2.jpg` — Sarah Chen (600×800px, portrait)
- `trainer-3.jpg` — Marcus Thorne (600×800px, portrait)
- `trainer-4.jpg` — Priya Sharma (600×800px, portrait)

### images/gallery/
- `gym-1.jpg` through `gym-9.jpg` (square or portrait, ~800×800px)

## Tips
- Use high-quality, professionally shot images for best results
- Dark/moody images work best with the black+yellow theme
- Compress images with tools like TinyPNG before adding
- Name files exactly as shown above
